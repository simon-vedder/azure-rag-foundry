Contoso Technologies — Company Overview
========================================

About Contoso Technologies
---------------------------
Contoso Technologies is a mid-sized enterprise software company founded in 2008, headquartered in Stockholm, Sweden. We employ approximately 1,200 people across offices in Stockholm, Berlin, London, and Singapore. Our core business is enterprise resource planning (ERP) software for manufacturing and logistics companies across the EMEA region.

Our Products
-------------
1. ContosoERP — Our flagship ERP platform used by over 400 enterprise customers. It covers finance, procurement, inventory, production planning, and HR modules. Latest version: 12.4 (released March 2026).

2. ContosoLogistics — A supply chain management add-on for ContosoERP. Integrates with major logistics providers (DHL, FedEx, DB Schenker) and provides real-time shipment tracking.

3. ContosoAnalytics — A business intelligence layer built on top of ContosoERP data. Uses Microsoft Azure for data warehousing and Power BI for dashboards.

Company Strategy 2026–2028
----------------------------
Contoso's three-year strategy is focused on three pillars:
1. Cloud-first migration — Moving all customers from on-premise ContosoERP to ContosoERP Cloud (Azure-hosted SaaS) by end of 2028.
2. AI integration — Embedding generative AI capabilities into the ERP platform, starting with automated purchase order processing and anomaly detection in financials.
3. Geographic expansion — Entering the North American market by establishing a new office in Toronto, Canada, planned for Q3 2026.

Technology Stack
-----------------
- Backend: Java (Spring Boot), Python microservices for AI features

- Cloud: Microsoft Azure (primary), on-premise (legacy customer deployments)
- Data: Azure SQL Database, Azure Data Lake Storage, Apache Kafka for event streaming
- AI/ML: Azure OpenAI Service, Azure Machine Learning
